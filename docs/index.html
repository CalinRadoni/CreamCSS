The purpose of this framework is to be used as a starting point for web pages. It is Lightweight and Classless so it can be used by adding to a page's header: 

<link rel="stylesheet" href="cream.css">

to turn the default ugly formatting into a beautiful page.

The framework was designed by following some design patterns from:

  • Google's Material Design 3
  • GitHub's Primer Design System
  • Apple's Human Interface Guidelines

To be sure that the look is the same in different browsers modern-normalize should be included before CreamCSS. modern-normalize can be downloaded from GitHub, use the minified version for production.

To continue with typography, this paragraph have a bottom border then follows a horizontal line and a blockquote.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Provident voluptas doloribus esse, enim, consequatur aliquam, saepe porro mollitia molestias recusandae itaque quibusdam? Nihil tempore provident deserunt nemo ea odit voluptates.

Custom elements may complicate the usage and increase the framework's size, custom attributes should start with data- and increase the framework's size in the long term so, for added flexibility, I have decided to add some classes:

  • c-req to style the labels of required inputs
  • c-main to apply the primary color to an element
  • border-b, defined in cream-extra.css file, to set the bottom border of an element

There are some custom HTML semantic elements in the cream-extra.css file:

  • cream-row
  • cream-card

Include that file if you want to use those.

Here is a complete <head>:

<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
  <meta name="description" content="Page description">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="modern-normalize.css">
  <link rel="stylesheet" href="cream.css">
  <link rel="stylesheet" href="cream-extra.css">
</head>

Colors

All primary, surface and outline color are derived from a single hue value.

  • The primary colors are with 90% saturation.
  • The surface and outline colors are with 10% saturation.
