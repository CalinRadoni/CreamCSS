# CreamCSS

CSS framework uses `HTML Semantic Elements` and is built to be:

- `Lightweight`
- `Classless`*
- `Responsive`
- `Mobile First`

## Documentation

Documentation and demo page is located in [CreamCSS](http://CalinRadoni.github.io/CreamCSS)
